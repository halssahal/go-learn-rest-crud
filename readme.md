# Learn CRUD Restful API Go-Lang

This restful is built using [go 1.18](https://go.dev)


### Run
```
$ cd $GOPATH/src/go-learn-rest-crud
$ go run main.go 
```

### Features

- FindTasks
- CreateTask
- FindTask
- UpdateTask
- DeleteTask
